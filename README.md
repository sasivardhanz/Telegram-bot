# Telegram Bot 🤖

This is a simple Telegram bot built using Python and deployed on Railway.

## 🚀 Features
- Responds to /start command
- Runs continuously using Railway
- Uses environment variables for security

## 🛠 Tech Stack
- Python
- pyTelegramBotAPI
- Railway (Deployment)

## 🔐 Setup

1. Create a bot using BotFather on Telegram
2. Get your BOT_TOKEN
3. Add it in Railway Variables:

BOT_TOKEN=your_token_here

## ▶️ Run Locally

pip install -r requirements.txt  
python main.py

## ☁️ Deployment

Connected with GitHub and deployed using Railway.

## 📌 Note
Do not expose your bot token publicly.
